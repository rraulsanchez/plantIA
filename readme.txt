# Documentación y plantilla del Proyecto TFG – Análisis de la IA para el desarrollo de aplicaciones en perfiles no programadores

## 📘 Cómo usar esta documentación

   Este proyecto explora cómo personas sin conocimientos profundos de programación pueden usar herramientas de **Inteligencia Artificial (IA)** para crear sitios web sencillos, con un caso práctico centrado en la generación de 
   landing pages o maquetas web (mockups) orientadas al sector turístico. Aunque el caso práctico se centra en el sector turístico, el sistema es fácilmente adaptable a otros contextos como tiendas online, páginas personales o 
   negocios locales de cualquier ámbito. El objetivo a groso modo es que cualquier persona, sin necesidad de conocimientos técnicos, pueda generar su propia web de forma sencilla y asistida por inteligencia artificial.

   Puedes revisar cada archivo por separado o seguir el orden sugerido para comprender el funcionamiento completo del proyecto.

   Aquí encontrarás todos los archivos que explican el funcionamiento del proyecto y cómo está organizado.

---

## Archivos incluidos y su propósito

   📁 documentacion-proyecto/
      -> Es la parte técnica que explica cómo funciona todo por dentro. Es útil para ti, tu tutor, o alguien que quiera mejorar o evaluar el sistema. No es necesario que el usuario final lea esto.
      │
      ├── 01-descripcion-general.md           ← ¿Qué es esto? ¿Por qué es útil? ¿A quién ayuda?
      ├── 02-requisitos-del-sistema.md        ← ¿Qué tiene que hacer el sistema mínimo para funcionar bien?
      ├── 03-estructura-tecnica.md            ← Cómo está montado internamente (sin meterse en complicaciones de código).
      ├── 04-funcionalidades.md               ← Qué puede hacer la plataforma (crear secciones, conectar a Google Maps, etc.).
      ├── 05-implementacion-con-ia.md         ← Cómo la IA genera contenido automáticamente.
      ├── 06-metodologia.md                   ← Cómo se organizó el trabajo (ej. por fases, como en un proyecto real).
      ├── 07-consideraciones-legales.md       ← Privacidad, licencias, derechos de imagen, etc.
      ├── 08-glosario-basico.md               ← Diccionario simple de palabras clave como "plantilla", "hosting", "frontend"...
      └── 09-ejemplos-de-uso-avanzado.md      ← Otras aplicaciones futuras: webs personales, tiendas, eventos...

   📁 plantilla-usuario/
         -> Es lo que ve y toca el usuario. Aquí es donde puede rellenar su información (nombre del negocio, secciones que quiere, colores...), y la IA lo usará para generar su web. Debe ser fácil, guiado y sin lenguaje técnico.
      │
      ├── instrucciones.md                    ← Guía básica paso a paso: "¿Qué hago primero?"
      ├── plantilla-web.md                    ← El archivo que el usuario debe rellenar (como si fuera un formulario).
      ├── ejemplo-relleno.md                  ← Una plantilla ya completa como ejemplo.
      ├── opciones-avanzadas.md               ← Extras opcionales: colores, idiomas, estructura personalizada.
      ├── flujo-visual.md                     ← Imagen o esquema simple del paso a paso desde que empiezan hasta publicar.
      └── preguntas-frecuentes.md             ← Respuestas a dudas comunes (“¿Qué pasa si no tengo fotos?”, “¿Puedo editar después?”)
